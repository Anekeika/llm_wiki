# wiki/

> ⚠️ **Возможно дубликат с `vault/wiki/`.** Wiki по проектам сейчас ведутся внутри Obsidian vault: `vault/wiki/<project>/`. Эта папка — для **публикуемой** wiki, если когда-нибудь понадобится отделить публичное от приватного.

## Когда эта папка пригодится

- **MkDocs / Docusaurus / Astro Starlight site** — генератор статического сайта из Markdown. Положить сюда исходники → собрать → задеплоить на GitHub Pages / Vercel.
- **Публичная версия** части vault'а — например, экспортируешь сюда отдельные wiki/peon-ping/ (открытый проект) без личного контента.
- **Cross-project wiki** — общие концепции которые не привязаны к конкретному проекту и не должны быть в Obsidian (например, шпаргалки по командам которыми ты делишься с коллегами).

## Что сейчас

Пусто. Если решишь что эта папка избыточна — удали её, ничего не сломается.

## Что НЕ сюда

- Личная wiki по проектам → `vault/wiki/<project>/`
- Дейли-логи → `vault/daily/`
- Скомпилированная база знаний → `vault/knowledge/`
