# anekeika LLM Wiki — Метод Карпати

Вики по устройству LLM, построенная по методологии Андрея Карпати:
**понять через реализацию** — каждая концепция объясняется с нуля с кодом.

## Структура

```
wiki/          # статьи по темам (Markdown)
code/          # минимальные реализации «с нуля»
notebooks/     # Jupyter-тетради с экспериментами
assets/        # схемы, изображения
```
