# assets/

Бинарные ассеты которые не должны лежать внутри `vault/`: чтобы Obsidian-граф не засорялся не-markdown файлами и vault оставался чисто "знания в Markdown".

## Что сюда класть

- Скриншоты и диаграммы для README/SETUP
- Иконки/лого проекта
- Экспортированные PDF (если делаешь презентацию/отчёт)
- Мокапы интерфейсов
- Аудио/видео из обучающих материалов

## Что НЕ сюда

- Картинки внутри статей wiki → в `vault/raw/Clippings/` или рядом с самой статьёй (Obsidian сам управляет attachments)
- Иконки приложений из проектов → в репо проекта, не сюда
