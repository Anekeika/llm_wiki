# CLAUDE.md — Схема wiki

Этот файл определяет как Claude ведёт и поддерживает wiki.
Цель wiki: накапливать контекст по проектам между сессиями, чтобы Claude не начинал с нуля.

## Структура vault

```
wiki/<project>/       # wiki по проекту
  index.md            # каталог + краткое состояние проекта
  decisions/          # архитектурные решения
  concepts/           # ключевые концепции проекта
  qa/                 # сохранённые ответы на вопросы
raw/                  # исходники (иммутабельно)
  Clippings/          # из Obsidian Web Clipper
log.md                # хронологический лог всех операций
```

## Старт сессии

1. Прочитать `wiki/<project>/index.md` нужного проекта
2. Сверить 2–3 ключевых факта с текущим кодом
3. Обновить index.md если что-то устарело

## Конвенции именования файлов

- Решения: `decisions/название-решения.md`
- Концепции: `concepts/название.md`
- Q&A: `qa/вопрос-кратко.md`
- Источники (raw): оригинальное имя файла, не переименовывать

## Frontmatter (обязательно для всех wiki-файлов)

```yaml
---
title: "Название"
project: "<project-slug>"
created: YYYY-MM-DD
updated: YYYY-MM-DD
tags: []
---
```

## Что фиксировать обязательно

- Архитектурные решения с обоснованием (почему так, а не иначе)
- Нетривиальные паттерны и gotchas
- Решённые неочевидные баги
- Ответы на вопросы которые возникали больше одного раза

## Что НЕ фиксировать

- Тривиальные правки (опечатки, косметика)
- Временные решения которые будут переделаны
- Детали которые очевидны из кода

## index.md — структура

```markdown
# <Project> — index

## Состояние проекта
Одна строка: что сейчас в работе, последний значимый PR.

## Стек
Краткий список технологий.

## Ключевые файлы
- `путь/к/файлу` — для чего

## Решения
- [[decisions/название]] — одна строка суть

## Открытые вопросы
- Список того что неясно или в процессе
```

## Операции

**Ingest:** зафиксировать решение/блок работы
**Query:** ответить на вопрос по wiki, опционально сохранить в qa/
**Lint:** проверить противоречия, orphans, устаревшее — раз в неделю

## Конвенция коммитов

```
wiki(<project>): ingest <что зафиксировано>
wiki(<project>): lint weekly
wiki: add <project> index
schema: <что изменено в CLAUDE.md>
```
