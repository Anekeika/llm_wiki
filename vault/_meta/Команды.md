# Команды

---

## Команды для Claude

Говори это в любой сессии — Claude поймёт что речь о wiki:

| Что сказать | Что произойдёт |
|-------------|----------------|
| `/wiki ingest` | Зафиксировать текущее решение / блок работы в wiki проекта |
| `/wiki query <вопрос>` | Найти ответ в wiki и показать |
| `/wiki query <вопрос> --save` | Найти ответ и сохранить страницей в wiki |
| `/wiki lint` | Проверить здоровье wiki текущего проекта |
| `/wiki status` | Показать что сейчас в wiki: проекты, кол-во страниц, последняя запись в log |
| `/wiki open <проект>` | Прочитать index.md нужного проекта |

## Скрипты memory-compiler

Запускать из `E:\anekeika-llm-wiki\memory-compiler\`:

```bash
# Скомпилировать daily logs → wiki-статьи
uv run python scripts/compile.py

# Вопрос к wiki
uv run python scripts/query.py "вопрос"

# Вопрос к wiki + сохранить ответ в wiki
uv run python scripts/query.py "вопрос" --file-back

# Полная проверка здоровья wiki (7 проверок, использует API)
uv run python scripts/lint.py

# Только структурные проверки (бесплатно, без API)
uv run python scripts/lint.py --structural-only
```

## Хуки (автоматически, ничего делать не нужно)

| Хук | Когда срабатывает | Что делает |
|-----|-------------------|-----------|
| `SessionStart` | Начало каждой сессии | Читает `knowledge/index.md` → вставляет в контекст |
| `SessionEnd` | Конец сессии | Захватывает транскрипт → `daily/YYYY-MM-DD.md` |
| `PreCompact` | Перед авто-сжатием контекста | Захватывает всё до потери деталей |
