# code/

Экспериментальный/прототипный код. **НЕ движок памяти** — он живёт в `memory-compiler/`.

## Что сюда класть

- Прототипы новых хуков перед тем как ввести в `memory-compiler/`
- Одноразовые скрипты (миграция данных, экспорт, конвертация vault → JSON и т.д.)
- Сниппеты которые упоминаются в wiki-статьях ("вот пример как я делал X")
- Эксперименты с Claude Agent SDK не для прода
- Утилиты обработки `vault/daily/*.md` (статистика, графики)

## Структура

Свободная — каждый эксперимент в своей подпапке с README:
```
code/
  vault-stats/          # подсчёт токенов в дейли
    README.md
    main.py
  obsidian-export/      # экспорт vault в HTML
    ...
```

## Что НЕ сюда

- Production-код движка → `memory-compiler/`
- Setup-скрипты → корень репо (`setup.ps1`)
